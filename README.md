# go-tetris

A simple Tetris game written in Go.

## Installation
```
go get https://github.com/rhallman96/go-tetris
```

## Controls

| Action       | Key         |
|--------------|-------------|
| Rotate Left  | Z           |
| Rotate Right | X           |
| Move Left    | Left Arrow  |
| Move Right   | Right Arrow |
| Speed Up     | Down Arrow  |
| Quick Drop   | Up Arrow    |
